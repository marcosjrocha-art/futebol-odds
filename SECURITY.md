# Política de Segurança

Este projeto é educacional e não lida com dados sensíveis de usuários.

## Reportando vulnerabilidades
Se você identificar algum problema de segurança ou uso indevido:
- Abra uma Issue no GitHub descrevendo o problema
- Ou entre em contato diretamente com o mantenedor do repositório

Nenhuma informação pessoal é coletada ou armazenada por este sistema.
