# Contribuindo com o Futebol Odds Platform

Obrigado por considerar contribuir com este projeto!

## Diretrizes
- Este é um projeto educacional e estatístico
- Não serão aceitas funcionalidades que prometam lucro ou incentivo a apostas
- Código deve ser claro, comentado e reproduzível

## Sugestões bem-vindas
- Melhorias de desempenho
- Novos mercados estatísticos (bem justificados)
- Melhorias na calibração e métricas
- UI/UX e acessibilidade

## Como contribuir
1. Faça um fork do repositório
2. Crie uma branch (`feature/minha-melhoria`)
3. Faça commits claros
4. Abra um Pull Request explicando a mudança
